## Morse sequences as huffman code

    Consider the sequences of huffman code with size 4
    and correspondent morse code, 1(-) and 0(.)
                
    f   1101    Q
    h   1010    C
    i   1000    B
    m   0111    J
    n   0010    F
    s   1011    Y
    t   0110    P
